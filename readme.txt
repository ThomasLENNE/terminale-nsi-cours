Ces activités ont été rédigées à l'aide de jupyter notebook.

Les documents à ouvrir sont les fichiers au format .ipynb.

Dans chaque dossier, un lien 'binder badge' en bas de page permet d'accéder à ce fichier en ligne

Sinon rappel de la procédure pour ouvrir ce type de fichier une fois téléchargé sur votre machine :
* ouvrir une console de commande à l'emplacement du fichier.
* taper jupyter notebook ou si cela ne fonctionne pas :  python -m notebook


Pour les activités débranchées, on pourra aussi utiliser les fichiers au format .pdf
